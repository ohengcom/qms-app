## ✨ 核心功能

### 1. 被子信息管理 🛏️

**完整的 CRUD 操作**:

- 添加、编辑、删除被子信息
- 智能搜索和筛选（季节、状态、颜色、材质）
- 列表视图和网格视图无缝切换
- 批量操作支持
- 双击行快速操作（可配置）

**智能命名系统**:

- 自动生成被子名称：品牌 + 颜色 + 重量 + 季节
- 示例：`百思寒褐色1100克春秋被`

**字段管理**:

- 基本信息：名称、季节、尺寸、重量、材料
- 存储信息：位置、包装、品牌
- 状态管理：使用中、存储中、维护中

### 2. 使用记录跟踪 📊

**自动化追踪**:

- 状态改为"使用中"时自动创建使用记录
- 状态改为其他时自动结束活动记录
- 可选择自定义开始/结束日期
- 支持添加备注

**使用历史**:

- 查看单个被子的完整使用历史
- 编辑和删除使用记录
- 使用时长统计
- 温度信息显示

**智能状态管理**:

- 三种状态智能转换
- 状态改为"使用中"时自动设置位置为"在用"
- 防止重复创建使用记录

### 3. 数据分析 📈

**四大分析模块**:

1. **数据概览**
   - 总体统计数据
   - 季节使用情况
   - 快速数据洞察

2. **状态分布**
   - 被子状态分布可视化
   - 季节分布图表
   - 实时数据更新

3. **使用排行**
   - 最常用被子排行榜
   - 年度使用趋势
   - 使用次数统计

4. **使用频率分析**
   - 智能分析使用频率
   - 提供优化建议
   - 数据驱动决策

### 4. 导入导出 📥📤

**灵活的数据交换**:

- Excel 文件导入（.xls, .xlsx）
- CSV 和 JSON 格式导出
- 完美支持中文
- 数据备份和迁移

### 5. 系统设置 ⚙️

**个性化配置**:

- 双击行为配置（无操作/状态/编辑）
- 密码修改（数据库存储）
- 应用名称自定义
- 语言切换（中文/英文）
- 系统信息查看

### 6. 认证与安全 🔐

**安全保护**:

- 密码保护登录
- JWT 会话管理
- 路由保护中间件
- 会话持久化（记住我）
- 安全 Cookie 存储

## 🏗️ 技术架构

### 前端技术栈

- **框架**: Next.js 16.0.0 (App Router)
- **语言**: TypeScript 5.6.3
- **样式**: Tailwind CSS 4
- **UI 组件**: Radix UI (Shadcn UI)
- **动画**: Framer Motion
- **状态管理**: Zustand, React Query
- **表单**: React Hook Form + Zod

### 后端技术栈

- **数据库**: Neon Serverless PostgreSQL
- **API**: Next.js API Routes + tRPC
- **认证**: JWT + bcryptjs
- **验证**: Zod schemas

### 开发运维

- **部署**: Vercel
- **版本控制**: Git + GitHub
- **代码质量**: ESLint, Prettier
- **包管理**: npm

## 🚀 快速开始

### 在线体验

访问 [https://qms-app-omega.vercel.app](https://qms-app-omega.vercel.app) 立即体验

### 本地部署

```bash
# 克隆仓库
git clone https://github.com/ohengcom/qms-app.git
cd qms-app

# 安装依赖
npm install

# 设置环境变量
cp .env.example .env.local
# 编辑 .env.local 填入你的 Neon 数据库 URL

# 设置管理员密码
npm run setup-password

# 启动开发服务器
npm run dev
```

访问 http://localhost:3000
