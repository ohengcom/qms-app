# 🎉 被子管理系统 v1.0.0 - 首个稳定版本

这是被子管理系统（QMS）的首个稳定版本！经过多次迭代和优化，系统已经具备完整的功能和良好的用户体验。

## ✨ 主要功能

### 被子信息管理

- 完整的CRUD操作（添加、编辑、删除、搜索）
- 支持多种筛选条件（季节、状态、颜色、材质）
- 列表视图和网格视图切换
- 批量操作支持
- 双击行快速操作

### 使用记录跟踪

- 自动记录被子使用历史
- 查看单个被子的详细使用记录
- 编辑和删除使用记录
- 使用时长统计
- 温度信息显示

### 数据分析

- **数据概览**：总体统计数据和季节使用情况
- **状态分布**：被子状态和季节分布可视化
- **使用排行**：最常用被子排行榜和年度趋势
- **使用频率分析**：智能分析使用频率，提供优化建议

### 导入导出

- Excel文件导入（.xls, .xlsx）
- CSV和JSON格式导出
- 数据备份和迁移

### 系统设置

- 双击行为配置
- 密码修改
- 应用名称自定义
- 语言切换（中文/英文）
- 系统信息查看

## 🎨 UI优化亮点

- 统一使用 Shadcn UI 组件库，界面更加现代化
- 优化表格样式，统一排序图标和操作列
- 改进空状态和错误提示显示
- 响应式设计，完美支持移动端
- 流畅的动画效果和交互体验

## 📊 技术栈

- **前端框架**: Next.js 14 (App Router)
- **语言**: TypeScript
- **数据库**: Prisma + Neon PostgreSQL
- **样式**: Tailwind CSS
- **UI组件**: Shadcn UI
- **状态管理**: Zustand + TanStack Query
- **API**: tRPC (类型安全的API调用)

## 🚀 快速开始

### 在线体验

访问 [https://qms-app.vercel.app](https://qms-app.vercel.app) 立即体验

### 本地部署

```bash
# 克隆仓库
git clone https://github.com/ohengcom/qms-app.git
cd qms-app

# 安装依赖
npm install

# 配置环境变量
cp .env.example .env.local
# 编辑 .env.local 填入数据库连接信息

# 运行开发服务器
npm run dev
```

访问 http://localhost:3000
